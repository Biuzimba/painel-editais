⚙ Configurar Firebase

Como configurar em 5 minutos
1. Acesse console.firebase.google.com e crie um projeto gratuito.
2. Vá em Firestore Database → Criar banco de dados (modo de teste).
3. Vá em Configurações do Projeto → Seus aplicativos → Web → Adicionar app.
4. Copie as credenciais do firebaseConfig e cole abaixo.
5. No Firestore, vá em Regras e aplique: allow read, write: if true; (para uso interno).
⚠ As regras de leitura/escrita abertas são adequadas para uso interno do time. Para uso externo, configure autenticação.
